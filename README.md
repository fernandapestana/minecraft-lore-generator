# minecraft-lore-generator

Gerador de lores medievais para itens RPG e Minecraft utilizando HTML, CSS e JavaScript.

O projeto gera aleatoriamente nomes, raridades e descrições (lore) para itens medievais inspirados em universos RPG e Minecraft.
Cada categoria possui diferentes combinações pré-definidas, permitindo gerar itens únicos de forma dinâmica.

Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

Funcionalidades

- Seleção de tipos de itens
- Geração aleatória de lores
- Sistema de raridade
- Interface medieval temática
- Botão para copiar lore

Objetivo

Este projeto foi criado para uma entrega da faculade e comm ele pude praticar:
- lógica de programação;
- manipulação de DOM;
- randomização em JavaScript;
- desenvolvimento de interfaces temáticas.

Tema

Inspirado em:
- Minecraft
- RPG medieval
- Sistemas de itens e raridades
